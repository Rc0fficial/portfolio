# Agile IT Blog App

<Image src={agileit} alt="Agile IT Blog App" />

Maintained a blog app developed in Astro.js, Markdown, Tailwind CSS, CSS, SCSS, with modular components and data integrations. Collaborated with a team to improve the app's performance and scalability.

## Description

- Developed a blog app using Astro.js, combining the power of static site generation with dynamic capabilities.
- Utilized Markdown for content creation, enabling easy and flexible post management.
- Designed a responsive and visually appealing user interface with Tailwind CSS and SCSS for custom styling.
- Implemented modular components for a scalable and maintainable codebase.
- Integrated various data sources and APIs to provide rich and up-to-date content to users.

## Technologies Used

- **Astro.js**
- **Markdown**
- **Tailwind CSS**
- **SCSS**
- **Modular Components**
- **Data Integrations**
