# Dare To Donate — Blood Donation App

<Image src={dareToDonate} alt="Dare To Donate Blood Donation App" />

Independently developed a blood donation app using React Native, Redux Toolkit, and Firebase, utilizing React Native Expo for the development process. This project aimed to facilitate blood donation by providing an easy-to-use platform for donors and recipients.

## Description

- **React Native:** Used React Native to create a cross-platform mobile application, ensuring a consistent user experience on both iOS and Android devices.
- **Redux Toolkit:** Implemented Redux Toolkit for state management, allowing for efficient and scalable handling of application state.
- **Firebase:** Leveraged Firebase for backend services, including authentication, real-time database, and cloud storage, ensuring secure and reliable data handling.
- **React Native Expo:** Utilized React Native Expo for streamlined development, enabling easy deployment and testing of the app.
- **User Interface:** Designed a user-friendly interface to simplify the process of finding and donating blood, enhancing accessibility and engagement.
- **Independent Development:** Managed the entire development process independently, from initial design to deployment, demonstrating strong self-management and problem-solving skills.
