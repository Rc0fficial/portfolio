# Gatsby App With Strapi — Custom Web App

<Image src={brain} alt="Gatsby App With Strapi" />

Developed a custom web app using Gatsby and Tailwind CSS, featuring a fully integrated blog powered by Strapi. This project combines fast performance with a sleek design, offering a seamless user experience for content creation and consumption.

## Description

- **Gatsby:** Utilized Gatsby for static site generation, ensuring fast performance and optimal SEO.
- **GraphQL:** Implemented GraphQL queries to fetch data efficiently, providing a flexible and powerful data management solution.
- **Tailwind CSS:** Used Tailwind CSS for styling, enabling rapid development of a responsive and modern UI. Applied utility-first classes to ensure a clean and maintainable codebase.
- **Strapi:** Integrated Strapi as the headless CMS, allowing for easy content management and API creation.
- **Multiple Categories:** Designed the app to support multiple categories, each with a unique UI, enhancing content organization and user navigation.
- **Custom Blog Features:** Developed a fully integrated blog with features such as category-specific pages, dynamic content rendering, and customizable layouts.
- **User Experience:** Focused on creating a seamless user experience for both content creators and consumers, ensuring ease of use and aesthetic appeal.
