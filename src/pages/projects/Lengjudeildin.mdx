## Description:
I worked with a team on an advanced project focused on automating football league operations using React and GraphQL Apollo Client. This automated app provides real-time updates and data management for football leagues, significantly enhancing user engagement.

The project involved creating dynamic widgets that update in real-time, seamlessly integrating with live streams on the [YouTube Channel](https://www.youtube.com/@Lengjudeildin). These widgets automated the display of live scores, statistics, and other crucial information, improving the viewer experience.
